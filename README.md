🎮 Dungeon Game

Python roguelike-шутер на Pygame — зачисти подземелье от зомби, бандитов и босса, чтобы выбраться на следующий уровень.


📖 Описание
Пошаговый рогалик-шутер с процедурно генерируемыми подземельями. Игрок спускается всё глубже, зачищая комнаты от врагов. Выход из комнаты заблокирован, пока внутри есть хоть один враг. Переход на следующий уровень открывается только после полной зачистки всего подземелья.

🕹️ Геймплей

Каждая комната заперта — пока не убьёшь всех врагов внутри, дверь не откроется
Переход на следующий уровень возможен только после зачистки всего подземелья
Подземелья генерируются случайно при каждом запуске
При гибели игрок начинает заново — перманентная смерть (permadeath)


👤 Персонажи
На выбор доступны два персонажа с разными характеристиками:
персонаж 1: быстрый, больше хп, больше урон. персонаж 2: медленный, меньше хп, меньше урон

👾 Враги
🧟 Зомби
Медленные, но опасные в большом количестве. Идут напролом.
🔫 Бандиты
Быстрые и агрессивные. Преследуют игрока по кратчайшему пути.
💀 Босс
Появляется в финальной комнате уровня. Обладает повышенным запасом здоровья. Без его уничтожения подземелье не считается зачищенным.

🗂️ Структура проекта
├── src/
│   ├── dungeon/
│   │   └── generation_dungeon.py   # процедурная генерация лабиринтов
│   └── entity/
│       ├── bullet.py               # снаряды
│       ├── enemy.py                # базовый класс врагов
│       ├── enemy_type.py           # зомби, бандиты, босс
│       ├── items.py                # предметы
│       ├── weapon.py               # оружие
│       └── player.py               # игрок
├── config.py                       # константы
├── game.py                         # основной класс игры
├── renderer.py                     # отрисовка
├── handler.py                      # обработчик событий
├── world.py                        # объекты на карте
└── main.py                         # точка входа
